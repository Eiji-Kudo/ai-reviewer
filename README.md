# AI Reviewer

AI-powered code review dashboard for efficient and intelligent code review workflows.

## Features

- **AI-Powered Analysis** - Automatic code analysis with AI suggestions for improvements
- **Review Dashboard** - Track and manage all code reviews in one place
- **Kanban Board** - Visualize review status with drag-and-drop board view
- **Diff Viewer** - View file changes with syntax highlighting
- **AI Chat** - Interactive AI assistant for code review discussions
- **Comment Composer** - Generate review comments with adjustable tone (gentle/neutral/strict)
- **Dark Mode** - Full dark mode support with system preference detection

## Tech Stack

- React 19
- React Router 7
- TypeScript
- Tailwind CSS 4
- Vite 7
- Biome (linting/formatting)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### Build

```bash
npm run build
```

### Lint & Format

```bash
npm run lint        # Check for issues
npm run lint:fix    # Fix issues
npm run format      # Format code
```
